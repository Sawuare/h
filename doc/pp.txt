NAME
  pp - print a penis over time

SYNOPSIS
  pp [<length>]

DESCRIPTION
  If given, the decimal integer <length>, which must be greater than or equal to 3,
  specifies the character length of the penis, which is otherwise a random integer in
  the range [3, 85].

  pp pauses for 1/length of a second between characters.

EXIT STATUS
  1 if the argument is invalid;
  0 otherwise;
