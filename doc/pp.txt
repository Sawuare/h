NAME
  pp - print a penis incrementally

SYNOPSIS
  pp [<length>]

DESCRIPTION
  If given, the integer <length>, which must be in the range [3, 80], specifies the
  character length of the penis, which is otherwise a random integer in this range.

  pp pauses for 1/length of a second between characters.

EXIT STATUS
  1 if the argument is invalid;
  0 otherwise;
