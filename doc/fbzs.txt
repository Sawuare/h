NAME
  fbzs - write some interesting fb Z sequences

SYNOPSIS
  fbzs

DESCRIPTION
  The sequences are nonnegative integers less than 2 raised to 32, separated by
  spaces, and written to these files:

    a19n1.fb.txt
    a19p0.fb.txt
    a19p1.fb.txt
    a19p2.fb.txt
    a19p3.fb.txt

  Where aNN denotes a sequence for an fb painter with ID NN.

SEE ALSO
  fb2fb.txt, fb2png.txt
