NAME
  export.sh - export the output of some programs to ../dat/

SYNOPSIS
  export.sh

DESCRIPTION
  export.sh must be executed from ../src/. It exports the output of these programs,
  assuming that build.sh has been executed:

    fbzs.py
    getendian
    getlocale
    limits
    mts.py
    print.py
    waves
