NAME
  waves - write some fixed audio waves to WAVE files

SYNOPSIS
  waves

DESCRIPTION
  These are the waves and the files they are written to:

    Square wave   -> sqr.wav
    Triangle wave -> tri.wav

  Each wave has an amplitude of 0 dBFS, a duration of 1 s, and a frequency of 440 Hz.
  Each file has 1 channel, unsigned 8-bit samples, and 44100 samples per second.
