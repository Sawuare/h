NAME
  waves - write some fixed classic audio waves to WAVE files

SYNOPSIS
  waves

DESCRIPTION
  These are the waves and the files they are written to:

    Square wave   -> sqr.wav
    Triangle wave -> tri.wav

  Each wave has an amplitude of 127/128, a duration of 1 second, and a frequency of
  441 hertz. Each file has 1 channel, unsigned 8-bit samples, and 44100 samples per
  second.

EXIT STATUS
  1 if opening the file fails;
  2 if writing the file or closing it fails;
  0 otherwise;
