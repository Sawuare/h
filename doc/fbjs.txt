NAME
  fbjs - write some interesting fb j sequences

SYNOPSIS
  fbjs

DESCRIPTION
  The sequences are nonnegative integers less than 2 raised to 32, separated by
  spaces, and written to these files:

    a19n1.fb.txt
    a19p0.fb.txt
    a19p1.fb.txt
    a19p2.fb.txt
    a19p3.fb.txt
    a19p4.fb.txt
    a19p5.fb.txt
    a19p6.fb.txt
    a19p7.fb.txt
    a19p8.fb.txt
    a19p9.fb.txt

  Where aNN denotes a sequence for a fb painter with ID NN.

  The first 5 sequences are these sequences in the On-Line Encyclopedia of Integer
  Sequences (OEIS):

    https://oeis.org/A059153
    https://oeis.org/A020522
    https://oeis.org/A171476
    https://oeis.org/A171499
    https://oeis.org/A171472

SEE ALSO
