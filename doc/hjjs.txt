NAME
  hjjs - write some HJ J-sequences

SYNOPSIS
  hjjs

DESCRIPTION
  For a description of HJ J-values, see the beginning of hj.txt.

  The sequences are nonnegative integers less than 2 raised to 32, separated by
  spaces, and written to these files:

    i22m0.hjjs.txt
    i22m1.hjjs.txt
    i22m2.hjjs.txt
    i22m3.hjjs.txt
    i22m4.hjjs.txt
    i22m5.hjjs.txt
    i22m6.hjjs.txt
    i22m7.hjjs.txt
    i22m8.hjjs.txt
    i22m9.hjjs.txt

  Where "i#" denotes a sequence for the HJ painter whose ID is #.

  The first four sequences are these sequences in the On-Line Encyclopedia of Integer
  Sequences (OEIS):

    https://oeis.org/A020522
    https://oeis.org/A171476
    https://oeis.org/A171499
    https://oeis.org/A171472

SEE ALSO
