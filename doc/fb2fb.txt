NAME
  fb2fb - paint fb images on the framebuffer

SYNOPSIS
  fb2fb [-# <base16-color>] [-c <letter-color>] [-i <id>] [-l] [-s <step>] [-z <z>]

DESCRIPTION
  The following keys can be used to navigate and control fb2fb.

    {  Decrements ID;
    }  Increments ID;
    [  Decrements Z;
    ]  Increments Z;
    i  Sets ID to the decimal nonnegative integer entered next;
    z  Sets Z to the decimal positive integer entered next;
    l  Toggles the status line;
    e  Exits;

OPTIONS
  -# <base16-color>
    The hexadecimal nonnegative integer <base16-color> specifies the painting color
    in the form rrggbb.
    The default is white.

  -c <letter-color>
    The letter <letter-color> specifies the painting color and should be one of
    (r|g|b|c|m|y) for red, green, blue, cyan, magenta, or yellow.
    The default is white.

  -i <id>
    The decimal nonnegative integer <id> specifies the painter's ID and must be less
    than 33.
    The default is 0.

  -l
    Turn on the status line.
    The default is off.

  -s <step>
    The decimal nonnegative integer <step> specifies the Z decrement/increment step.
    The default is 1.

  -z <z>
    The decimal positive integer <z> specifies the painter's Z.
    The default is 1.

EXIT STATUS
  1 if an option is invalid;
  2 if opening the framebuffer device failed;
  3 if mapping the framebuffer device failed;
  4 if ID or Z is invalid;
  0 otherwise;
