NAME
  amorse - encode input into audible International Morse code

SYNOPSIS
  amorse [-d] [-f <frequency>] [-q] [-u <unit>]

DESCRIPTION
  The easiest way to use the program is to redirect morse output to amorse input.

OPTIONS
  -d
    Delete the file after trying to play it.

  -f <frequency>
    The decimal positive integer <frequency> specifies the signal frequency in hertz.
    The default is 400.

  -q
    Be quiet by not trying to play the file.

  -u <unit>
    The decimal positive integer <unit> specifies the unit duration in milliseconds.
    The default is 100.

EXIT STATUS
  1 if an option is invalid;
  2 if an option argument is invalid;
  3 if opening the file fails;
  4 if an input character is invalid;
  5 if the host environment has no command processor;
  0 otherwise;
